**Export 1: `validateModelId(modelId, apiKey)`** — async function per D-04.
- Create spinner via `p.spinner()`, call `s.start('Validating model ID with OpenRouter...')`
- `fetch('https://openrouter.ai/api/v1/models', { headers: { Authorization: \`Bearer ${apiKey}\` } })`
- Check `res.ok` before calling `res.json()` — if not ok, `s.stop(chalk.red(\`OpenRouter API error: ${res.status} ${res.statusText}\`))` and return false (per pitfall 3)
- Parse `body.data.some((m) => m.id === modelId)` for exact match
- On match: `s.stop('Model ID is valid.')` and return true
- On no match: `s.stop(chalk.red(\`Model "${modelId}" not found in OpenRouter model list.\`))` and return false
- Wrap entire fetch in try/catch — on error: `s.stop(chalk.red(\`Network error: ${err.message}\`))` and return false (per pitfall 1 — spinner must always be stopped)

**Export 2: `assignModel(tier)`** — async function, returns `true` on success, `false` on validation failure. Single-attempt function per research open question 1.
- Step 1: Get API key per D-05/D-06. Call `getOpenRouterKey()`. If null, prompt with `p.text({ message: 'Enter your OpenRouter API key:', placeholder: 'sk-or-...' })` with non-empty validate. On cancel: `p.cancel('Cancelled.')` + `process.exit(0)`. Trim and save via `saveOpenRouterKey(apiKey)`. This is identical to switch.js lines 10-31.
- Step 2: Prompt for model ID per D-01/D-02. Use `p.text({ message: \`Enter model ID for ${tier} tier:\`, placeholder: 'e.g. anthropic/claude-3-opus' })` with non-empty validate. On cancel: `p.cancel('Cancelled.')` + return true (exit gracefully, treat cancel as done).
- Step 3: Validate via `validateModelId(modelId.trim(), apiKey)`. If false, return false (per D-07 — caller will re-prompt).
- Step 4: Persist via `setModel(tier, modelId.trim())`. Log `p.log.success(chalk.green(\`${tier} tier set to: ${modelId.trim()}\`))`. Return true.

**Export 3: `assignModelByFlag(tier, modelId)`** — async function for CLI flag handlers per D-11.
- Step 1: Get API key (same D-05/D-06 pattern as assignModel step 1).
- Step 2: Validate via `validateModelId(modelId.trim(), apiKey)`. If false, `console.error(chalk.red(\`Model "${modelId.trim()}" not found on OpenRouter. Assignment cancelled.\`))` + `process.exit(1)`.
- Step 3: Persist via `setModel(tier, modelId.trim())`. Log `console.log(chalk.green(\`${tier} tier set to: ${modelId.trim()}\`))`.
- Note: No re-prompt loop for flags — invalid model exits with error code 1.

**Export 4: `runModelsMenu()`** — async function for interactive sub-menu per D-08/D-09.
- Call `getModels()` to get current assignments.
- Show `p.select({ message: 'Select a tier to configure:', options: [...] })` with four options:

  - `{ value: 'haiku', label: 'Haiku', hint: models.haiku || '(not set)' }`
  - `{ value: 'back', label: 'Back to main menu' }`
- On cancel or 'back': return.
- Otherwise, enter re-prompt loop per D-07: `let done = false; while (!done) { done = await assignModel(tier); }`
